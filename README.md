Installation

Linux/Termux\n
git clone https://github.com/Retr0-code/B0mbER_SMS/ \n
cd B0mbER_SMS/ \n
pip install -r requirements.txt \n
chmod +x B0mbER.py \n
python3 B0mbER.py \n

Windows
