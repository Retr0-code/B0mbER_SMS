#Installation

<b>Linux/Termux</b>

```sh
git clone https://github.com/Retr0-code/B0mbER_SMS/
```
```sh
cd B0mbER_SMS/
```
```sh
pip install -r requirements.txt
```
```sh
chmod +x B0mbER.py
```
```sh
python3 B0mbER.py
```
<br>

<b>Windows</b>

Download archive

Open the cmd in the folder where you unpacked the archive

```sh

```
